# 🎮 Tic-Tac-Toe Game (Python Tkinter)

A simple and interactive **Tic-Tac-Toe game** built using **Python** and **Tkinter**. This desktop application allows two players to play the classic Tic-Tac-Toe game with a graphical user interface.

## 📌 Features

- ✅ Interactive 3x3 game board
- ✅ Two-player gameplay (X and O)
- ✅ Automatic player turn switching
- ✅ Winner detection
- ✅ Winning combination highlighted in green
- ✅ Pop-up message when a player wins
- ✅ Simple and beginner-friendly code

## 🛠️ Technologies Used

- Python 3
- Tkinter (Python GUI Library)

## 🎯 How to Play

1. Player **X** starts the game.
2. Players take turns placing their symbol on the board.
3. The first player to align 3 symbols horizontally, vertically, or diagonally wins.
4. The winning cells turn green and a winner message is displayed.



## 🧠 Winning Conditions

The game checks these combinations:

- Horizontal:
  - [0,1,2]
  - [3,4,5]
  - [6,7,8]

- Vertical:
  - [0,3,6]
  - [1,4,7]
  - [2,5,8]

- Diagonal:
  - [0,4,8]
  - [2,4,6]
